Tipos de datos

Cuando programamos tenemos que trabajar con datos que nosotros entendamos y que el ordenador entienda

Hay tipos de datos bastantes frecuentes que sirva para representar una cadena de texto, que normalmente se indican entre comillas ""

Todos estos tipos de datos se tienen que acabar almacenándose para que la máquina lo pueda manejar

Las variables

Son contenedores que nos sirven para guardar éstos datos y utilizarlos.

En vez de escribir "Hola mundo!" vamos a poner: "estoy programando" que va a ser una variable

Para eso lo nombramos con una palabra reservada (dependiendo que lenguaje estemos usando) seguido del nombre de la variable.

  • var programacion = "¡Estoy programando!

Entonces la cadena de texto "¡Estoy programando! la almaceno en la variable llamada programacion que lo indico con el signo igual.

Si le quisiera cambiar la variable se tendría que volver a nombrar seguido de un igual y el cambio a realizar.

Por ejemplo: programacion = "Estoy cambiando el valor!

Los condicionales

Teniendo en cuenta los booleanos, algo que puede ser verdadero o falso, vamos a utilizarlos como deciciones que tenga que hacer la mácquina.

Que nuestro programa sólo puede realizar una acción en concreta si se dan las condiciones dadas.

  • var user = "MoureDev"

  • var password = "1234"

En éste momento entran los condicionales: lo que nos ayuda a que algo sea verdadero o falso.

Para eso tenemos:

  • if user = "MoureDev" && password = "1234" {

    print ("He accedido al sistema")

    }

Los iguales son dobles porque creamos una relación de igualdad.

el codigo dice que si el usuario y la contraseña son correctos se puede acceder al sistema.

En caso de que no se cumpla la condición se vuelve false.
